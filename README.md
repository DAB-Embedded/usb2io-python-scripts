# USB2IO python scripts

Python scripts for V2 and V3 of Python.

Info about USB2IO device: https://usb2io.com/

## Requirements

Using pip needs to install pyserial, numpy, scipy

```bash
pip install pyserial
```

## Usage

```bash
python hello.py /dev/ttyACM0
```
